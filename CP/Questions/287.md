### Trick:
- ind is a node and nums[ind] is next of that node.
- now, it will take form of a linked list.
- Follow [[Floydâ€™s Cycle detection or Hair Tortoise algorithm]]
- CAUTION: keep both slow and fast pointer at head initally (otherwise it will change the formula and algo).

### Code
```java
class Solution {
    public int findDuplicate(int[] nums) {
        int slow = 0;
        int fast = 0;

        do{
            slow = nums[slow];
            fast = nums[nums[fast]];
        }while(slow!=fast);

        slow = 0;
        while(slow!=fast){
            slow = nums[slow]; 
            fast = nums[fast];
        }

        return slow;
    }
}
```
