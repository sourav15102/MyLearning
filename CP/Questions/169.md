### Idea 1:
- If we sort the array, since it is gauranteed to have one element with frequency >(strictly) floor(length or arr)/2.
- `arr[length/2]` has to be the ans.

### Code:
```java
class Solution {
    public int majorityElement(int[] nums) {
        Arrays.sort(nums);
        int x = nums.length/2;
        return nums[x];
    }
}
```

### Code:
```java
class Solution {
    public int majorityElement(int[] nums) {
        int win = nums[0];
        int cnt=0;
        for(int n: nums){
            if(n==win){
                cnt++;
            }else{
                cnt--;
            }

            if(cnt<0){
                cnt = 1;
                win = n;
            }
        }

        return win;
    }
}
```
