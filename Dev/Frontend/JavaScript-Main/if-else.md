
if/else:
```js
if (num > 10) {
  return "Bigger than 10";
} else {
  return "10 or Less";
}

1   ==  1  // true
1   ==  2  // false
1   == '1' // true
"3" ==  3  // true


3 ===  3  // true
3 === '3' // false


1 !=  2    // true
1 != "1"   // false
1 != '1'   // false
1 != true  // false
0 != false // false
```
